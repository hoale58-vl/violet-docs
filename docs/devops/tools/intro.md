# Useful DevOps Tools

Essential tools and utilities for DevOps engineers and SREs.

## Overview

A collection of useful tools, utilities, and scripts that make DevOps work more efficient and productive.

## Categories

### Command Line Tools
- `jq`: JSON processor
- `yq`: YAML processor
- `kubectl`: Kubernetes CLI
- `helm`: Kubernetes package manager
- `terraform`: Infrastructure as Code
- `aws-cli`: AWS command line
- `gcloud`: Google Cloud CLI

### Debugging & Troubleshooting
- `curl`, `httpie`: HTTP clients
- `netcat`, `telnet`: Network testing
- `dig`, `nslookup`: DNS queries
- `tcpdump`, `wireshark`: Packet analysis
- `strace`, `ltrace`: System call tracing

### Performance & Monitoring
- `htop`, `top`: Process monitoring
- `iotop`: I/O monitoring
- `nethogs`: Network bandwidth
- `prometheus`: Metrics collection
- `grafana`: Visualization

### Container Tools
- `docker`: Container runtime
- `docker-compose`: Multi-container apps
- `dive`: Docker image exploration
- `hadolint`: Dockerfile linter
- `trivy`: Container vulnerability scanner

### Infrastructure
- `ansible`: Configuration management
- `packer`: Image building
- `vagrant`: Development environments
- `localstack`: Local AWS simulation

### CI/CD Tools
- GitHub Actions
- GitLab CI
- Jenkins
- CircleCI
- ArgoCD (GitOps)

## Productivity Tools

- **Terminal multiplexers**: tmux, screen
- **Shell enhancements**: oh-my-zsh, starship
- **Code editors**: vim, VS Code, nano
- **Git tools**: tig, lazygit, git-extras
