# Cloud Platforms

Cloud infrastructure, services, and best practices for AWS, GCP, and Azure.

## Overview

Modern cloud platforms provide scalable, reliable infrastructure and services for building and deploying applications globally.

## Covered Platforms

### AWS (Amazon Web Services)
- EC2, ECS, EKS
- Lambda, API Gateway
- S3, RDS, DynamoDB
- CloudFormation, CDK
- IAM and security

### GCP (Google Cloud Platform)
- Compute Engine, GKE
- Cloud Functions, Cloud Run
- Cloud Storage, Cloud SQL
- Deployment Manager
- IAM and security

## Common Services

- **Compute**: Virtual machines, containers, serverless
- **Storage**: Object storage, block storage, file storage
- **Databases**: Relational, NoSQL, in-memory
- **Networking**: VPC, load balancers, CDN
- **Security**: Identity management, encryption, compliance

## Best Practices

- Use Infrastructure as Code
- Implement proper IAM policies
- Enable monitoring and logging
- Design for high availability
- Optimize costs with right-sizing
- Follow the Well-Architected Framework
